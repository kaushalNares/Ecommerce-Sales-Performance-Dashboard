# 📊 Sales Performance & Customer Insights Dashboard

## 📌 Overview
This project presents an interactive **Sales Performance Dashboard** built using Microsoft Excel to analyze business data and generate actionable insights. The goal was to transform raw data into a structured and visually intuitive dashboard that supports data-driven decision-making.

---

## 🎯 Objectives
- Analyze overall sales performance using key business metrics  
- Understand customer behavior and purchasing patterns  
- Identify top-performing products and categories  
- Detect seasonal trends and high-performing time periods  
- Enable dynamic analysis using interactive filters  

---

## 📊 Key Features
- KPI cards for **Total Revenue, Orders, Customers, Products, and AOV**  
- Interactive slicers for filtering by **Date and Occasion**  
- Revenue analysis across **categories, products, cities, and days**  
- Monthly trend analysis to identify seasonal patterns  
- Clean and structured dashboard layout for easy interpretation  

---

## 🔍 Key Insights
- Revenue shows strong seasonal peaks in **February and August**  
- Certain product categories contribute significantly to total revenue  
- Difference between total orders and customers indicates **repeat purchase behavior**  
- Weekly trends highlight high-performing sales days  
- Top-performing products can support better inventory and marketing decisions  

---

## 🛠 Tools & Techniques Used
- **Microsoft Excel**  
- Pivot Tables  
- Data Cleaning & Transformation  
- Data Visualization  
- Dashboard Design
